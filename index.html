关于端口扫描

端口扫描是一种用于确定目标主机上哪些端口开放的技术。开放的端口可能代表系统上运行的服务，了解这些信息有助于评估网络安全性。

用途

关于密码生成器

强密码是保护您账户安全的基础。使用随机生成的复杂密码可以大大降低被破解的风险。本工具可以帮助您生成符合安全标准的随机密码。

安全建议

  • - 使用至少16位长度的密码
  • - 定期更换密码
  • - 不重复使用同一密码

密码强度

  • - 8-12位: 弱
  • - 12-16位: 中等
  • - 16位以上: 强

最佳实践

  • - 使用密码管理器
  • - 启用双因素认证
  • - 避免使用个人信息

关于 MD5

MD5 消息摘要算法是一种广泛使用的哈希函数，产生一个 128 位（16 字节）的哈希值（通常表示为 32 位十六进制数）。虽然 MD5 不再被认为是安全的加密哈希函数（存在碰撞漏洞），但它仍然广泛用于文件校验和其他不需要密码学安全性的应用场景。

用途

  • - 文件完整性检查
  • - 数据比较
  • - 非安全场景下的哈希

特点

  • - 128位哈希值
  • - 快速计算
  • - 不再推荐用于安全应用

示例

输入: "Hello World"

输出: 5eb63bbbe01eeed093cb22bb8f5acdc3

Base64 编码/解码工具

已复制!

关于 Base64

Base64 是一种基于 64 个可打印字符来表示二进制数据的编码方式，常用于需要通过文本协议传输二进制数据的场景，如在 URL、Cookie、网页中嵌入图像等。Base64 不是加密算法，只是一种编码方式，因此不提供数据安全性。

用途

  • - 邮件附件
  • - URL 安全传输
  • - 网页中嵌入图像

特点

  • - 使用 64 个字符
  • - 数据量增加约 33%
  • - 不是加密方法

示例

输入: "Hello World"

编码后: SGVsbG8gV29ybGQ=

关于 IP地址检测

验证IP地址是否符合 IPv4/IPv6 格式规范，并检测是否属于保留地址段（如内网地址、广播地址等）。

支持类型

  • - IPv4（如：192.168.1.1）
  • - IPv6（如：2001:0db8:85a3:0000:0000:8a2e:0370:7334）

检测内容

  • - 格式合法性
  • - 保留地址识别（如 127.0.0.0/8、::1 等）

示例

输入: 192.168.1.256
2001:0db8::1
输出: 192.168.1.256 (非法，IPv4地址段超出范围)
2001:0db8::1 (合法，公网地址)

使用说明

输入URL和请求头（JSON格式），选择请求方法，点击"生成代码"将自动生成对应的Python requests代码模板。

示例

输入:

URL: https://api.example.com/data

生成的代码:

import requests

url = "https://api.example.com/data"
headers = {
    "Authorization": "Bearer token123"
}

response = requests.get(url, headers=headers)
print(response.text)

关于 JSON 格式化

JSON（JavaScript Object Notation）是一种轻量级的数据交换格式，易于人类阅读和编写，同时也易于机器解析和生成。JSON 格式化工具可以将压缩的、难以阅读的 JSON 文本转换为格式化的、具有缩进和换行的易读格式，也可以将格式化的 JSON 压缩为最小尺寸。

用途

  • - 验证 JSON 格式

特点

  • - 语法高亮显示
  • - 错误检查

示例
